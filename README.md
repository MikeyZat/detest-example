# Example usage of the de-test testing library

## See it live

1. `git clone https://github.com/MikeyZat/detest-example.git`
2. `npm i`
3. `npm start`
4. `npm run detest`
5. `npm run another-detest`

Check out [detest.yaml config file](./detest.yaml) for a deeper understanding :)
